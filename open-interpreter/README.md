# Open interpreter

- Export `OPENAI_API_KEY`
- Run `interpreter -y`
- Input:

```
1. Download a YouTube URL https://www.youtube.com/watch?v=Oii_QX9kMTw
2. Cut the clip from 00:04-00:46
3. Transcribe it
4. Translate to Chinese
5. Convert .st to .ass
6. Burn the subtitles
```
