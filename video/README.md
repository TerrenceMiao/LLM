Video Summary
=============

![Video Summary](Video%20Summary.png)

- Setup
  - `Mac`
  - `Ollama` with **llama3.3** model, to run in local environment

```
$ conda create -n video python=3.11.11
$ conda activate video

$ pip install -r requirements.txt
```

```
$ brew install ffmpeg
```

- Run

```
$ python main.py
```
